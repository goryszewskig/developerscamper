# DevelopersCamper API

> Backed API for DevelopersCamper API, which is a bootcamp website

## Usage

Rename "config/config.env.example" to "config/config.env" and update env values

## Install dependencies

```
npm install
```

## Run App

```
#Run in dev mode
npm run dev

#Run app in prod mode
npm start
```
